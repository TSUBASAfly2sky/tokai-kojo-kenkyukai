🔗 GitHub連携設定（本番反映）

Personal Access Token（PAT）を登録すると、「保存する」ボタンを押した際に自動でGitHubの data.json を更新し、サイト全体に反映されます。

PATの作成手順：

  1. GitHubにログイン → 右上アイコン → Settings
  2. 左メニュー最下部 Developer settings → Personal access tokens → Tokens (classic)
  3. Generate new token (classic) → 名前を入力（例：tokai-kojo-admin）
  4. repo にチェック → Generate token → 表示されたトークンをコピー

📖 管理画面の使い方

【お知らせの追加】
「お知らせ管理」タブでタイトル・日付・内容を入力して「保存する」を押してください。

【活動報告の追加】
「活動報告管理」タブでタイトル・日付・場所・内容を入力します。内容は段落ごとに空行を入れると読みやすく表示されます。

【写真の追加】
活動報告フォームの「写真」エリアに、写真をドラッグ＆ドロップするか、クリックで選んでください。自動で軽く圧縮されてそのまま埋め込まれます。GitHubへ写真を別アップロードする必要はありません。サムネイルの「×」で削除できます。1報告あたり6枚程度までが快適です。

【公開・非公開の切り替え】
トグルスイッチで公開(緑)/非公開(グレー)を切り替えられます。非公開にするとサイトに表示されなくなります。

【編集・削除】
一覧の「編集」ボタンで内容を変更、「削除」で削除できます。

📨 申し込み・問い合わせの共有設定手順

フォームに届いた申し込みは、Googleスプレッドシートで役員全員が確認できます。

ステップ1:Googleフォームを作る

  1. forms.google.com でフォームを新規作成
  2. 項目例:お名前・連絡先メール・参加希望の見学会・備考などを追加
  3. 右上「送信」→「埋め込み」タブ→ iframeコードをコピーしてHP管理者に渡す

ステップ2:役員全員で内容を共有する

  1. フォームの「回答」タブ → スプレッドシートアイコンをクリックしてシートを作成
  2. 作成されたスプレッドシートを開く
  3. 右上「共有」→ 役員全員のGmailアドレスを追加し「編集者」権限で共有
  4. 役員全員が同じシートをリアルタイムで確認・返信内容の記入ができます

ステップ3:申し込みがきたらメールで自動通知する

  1. スプレッドシートのメニュー「ツール」→「通知」を選択
  2. 「新しい回答があるとき」にチェックを入れる
  3. 役員全員のGmailに自動でメールが届きます（複数名に設定可能）
