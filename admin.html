📖 管理画面の使い方

【お知らせの追加】
「お知らせ管理」タブでタイトル・日付・内容を入力して「保存する」を押してください。

【活動報告の追加】
「活動報告管理」タブでタイトル・日付・場所・内容を入力します。内容は段落ごとに空行を入れると読みやすく表示されます。

【公開・非公開の切り替え】
トグルスイッチで公開(緑)/非公開(グレー)を切り替えられます。非公開にするとサイトに表示されなくなります。

【編集・削除】
一覧の「編集」ボタンで内容を変更、「削除」で削除できます。

【GitHubへの反映方法】

  1. 右上の 📥 JSON書き出し をクリックして data.json をダウンロード
  2. GitHubのリポジトリで data.json を開く
  3. 鉛筆マーク（編集）→ ダウンロードしたファイルの中身を貼り付け
  4. 「Commit changes」ボタンで保存
  5. 数分後、公開サイトに反映されます

📨 申し込み・問い合わせの共有設定手順

フォームに届いた申し込みは、Googleスプレッドシートで役員全員が確認できます。またLINEに自動通知を送ることも可能です。

ステップ1:Googleフォームを作る

  1. forms.google.com でフォームを新規作成
  2. 項目例:お名前・連絡先メール・参加希望の見学会・備考などを追加
  3. 右上「送信」→「埋め込み」タブ→ iframeコードをコピーしてHP管理者に渡す

ステップ2:役員全員で内容を共有する

  1. フォームの「回答」タブ → スプレッドシートアイコンをクリックしてシートを作成
  2. 作成されたスプレッドシートを開く
  3. 右上「共有」→ 役員全員のGmailアドレスを追加し「編集者」権限で共有
  4. 役員全員が同じシートをリアルタイムで確認・返信内容の記入ができます

ステップ3:申し込みがきたらメールで自動通知する

  1. スプレッドシートのメニュー「ツール」→「通知」を選択
  2. 「新しい回答があるとき」にチェックを入れる
  3. 役員全員のGmailに自動でメールが届きます（複数名に設定可能）

ステップ4:LINEに自動通知を送る（希望する場合）

  1. IFTTT（無料）またはGoogle Apps Scriptを使用
  2. 「Googleスプレッドシートに新しい行が追加されたら」→「LINEにメッセージを送る」と設定
  3. LINE公式アカウントまたはLINEグループのMessaging APIで受信できます
  4. 設定が難しい場合は、HP管理者にお問い合わせください
